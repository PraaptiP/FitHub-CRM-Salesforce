# FitHub CRM - Salesforce Project

FitHub CRM is a Salesforce Lightning application designed to manage
fitness center operations.

## Features
- Member Management
- Trainer Management
- Class Scheduling
- Booking Management
- Payment Validation
- Reports and Dashboard

## Objects Used
- Member
- Trainer
- Class
- Booking

## Automation
- Formula Fields
- Validation Rules
- Reports
- Dashboard

## Tools Used
- Salesforce Lightning
- Custom Objects
- Lightning App Builder
